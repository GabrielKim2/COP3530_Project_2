<h2> Purpose </h2>
The purpose of this project is to look at Gainesville crime data and use it to determine where the safest apartment complexes are in Gainesville

<h2> How to Use </h2>
The only requirement for this code is that you have gnuplot installed on your computer. This is necessary for the plots to render.

Once gnuplot is installed, simply run the program either in CLion or through command line and enter the information it asks.
